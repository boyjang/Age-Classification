# Age-Classification
This is a Keras implementation of a CNN for estimating age and gender from a face image. Our data https://www.openu.ac.il/home/hassner/Adience/data.html#agegender. This git is based on University of Isarel paper www.openu.ac.il/home/hassner/projects/cnn_agegender


Using
Install machine learning libaries below: keras, tensorflow, open-cv. 
Train data with train_data.py. After that we have a model. 
Using that model by predict.py to predict age from an image
